# 0x00. C - Hello, World

This repository contains introductory projects in C programming focusing on foundational concepts like compiling, linking, and running C programs.

## Project Descriptions

- **0-preprocessor:** This project demonstrates the use of the C preprocessor (`gcc -E`) to understand how C code is processed before compilation.
  
- **1-compiler:** This project explores the compilation process in C using the `gcc` compiler to generate assembly code (`gcc -S`).

- **2-assembler:** Here, the assembly code generated in the previous step is assembled into machine code using the assembler (`gcc -c`).

- **3-name:** This project explores how executable files are created by linking object files together (`gcc` without any flags).

- **4-puts.c:** This project introduces the `puts` function in C to print a string to the standard output (`stdout`).

- **5-printf.c:** Here, the `printf` function is used to print formatted output to the standard output (`stdout`).

- **6-size.c:** This project demonstrates the use of the `sizeof` operator in C to determine the size of various data types.

## How to Use

Each subdirectory contains a `README.md` file with detailed instructions on how to compile and run the programs within that project. Follow the instructions provided in each project directory to understand the concepts covered and run the C programs successfully.
